
🏋️‍♂️ FitForge – Fitness Tracker Web Application
📌 Project Overview

FitForge is a modern fitness tracking web application that allows users to log workouts, monitor progress, and stay consistent with their training goals.

The application is built using React and Tailwind CSS, and it integrates with the WGER API to fetch real exercise data.

FitForge helps users:

Log workouts (exercise, sets, reps, weight, date)

View workout history

Track performance improvements over time

Stay organized and motivated

🛠️ Built With

React – Frontend framework

Tailwind CSS – Responsive styling

WGER API – Exercise data

Netlify / Vercel – Deployment

📦 Installation & Setup

Clone the repository:

git clone https://github.com/yourusername/fitforge.git

Navigate into the project directory:

cd fitforge

Install dependencies:

npm install

Start the development server:

npm run dev
